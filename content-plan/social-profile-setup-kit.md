# VitalBloom Blog Social Profile Setup Kit

Use this kit to create consistent social profiles that support the `VitalBloom Blog` brand entity.

## Core Identity

- Profile name: VitalBloom Blog
- Handle preference: vitalbloombog, vitalbloomblog, vitalbloom_blog, or vitalbloomdotblog
- Website: https://www.vitalbloom.blog/
- Contact email: hello@vitalbloom.blog
- Category: Health/Wellness Website, Educational Website, or Wellness Publication

## Short Bio

Evidence-informed wellness guides for nutrition, fitness, sleep, stress, mindfulness, and healthy living.

## Longer Bio

VitalBloom Blog publishes practical, evidence-informed wellness guides for real life. We cover nutrition, fitness, sleep, stress management, mindfulness, lifestyle routines, and healthy living checklists. Official site: https://www.vitalbloom.blog/

## Brand Description

VitalBloom Blog is the wellness publication at VitalBloom.blog. We publish free educational guides and checklists; we are not a supplement store, tea shop, skincare store, telemedicine service, or medical clinic.

## Profile Launch Checklist

- Use the same profile name everywhere: VitalBloom Blog.
- Add the official website URL exactly: https://www.vitalbloom.blog/
- Add the same short bio on every profile.
- Upload the same logo or simple brand image on every profile.
- Publish 3 starter posts before using the profile for outreach.
- Do not add social links to the website footer until the profiles exist and can be maintained.
